# TrimLog

Repositório oficial dos projetos web da **TrimLog**.

Este repositório contém duas aplicações:

* **site/** → Site institucional da TrimLog (`trimlog.com.br`)
* **lp/** → Landing Page de captação de franqueados (`lp.trimlog.com.br`)

---

# Estrutura do projeto

```
trimlog
├── site   # Site institucional (Vue)
└── lp     # Landing Page de franquia (Vue + Vite)
```

---

# Rodando os projetos localmente

Cada projeto possui suas próprias dependências.

## Rodar o site institucional

```
cd site
npm install
npm run serve
```

---

## Rodar a Landing Page

```
cd lp
npm install
npm run dev
```

---

# Build para produção

## Site institucional

```
cd site
npm run build
```

## Landing Page

```
cd lp
npm run build
```
